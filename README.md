SurviveRexburg
=================

A zombie survival game set in Rexburg, Idaho. Survive 30 days by exploring, scavenging, and fortifying your way through 56 real Rexburg locations.

## Play Now

[**Play SurviveRexburg**](https://churro777.github.io/SurviveRexburg/)

## History

Made the original game with [Hayley Cox](https://github.com/carissa888) for our CIT 260 class at BYU-Idaho in the spring of 2014. It was a Java Swing desktop app — the first time I learned about object oriented programming and Java, so don't judge that code too harshly.

## The 2026 Port

Now porting it to a modern web app built with React, TypeScript, Vite, and HTML5 Canvas. Pixel art retro aesthetic, playable in the browser on GitHub Pages.
